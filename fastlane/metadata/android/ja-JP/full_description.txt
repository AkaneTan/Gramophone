<p>Gramophone は .mp3、.flac、.ogg、.opus、.m4a、.mka、.wav ファイルが再生できるオフラインなローカル音楽プレーヤーです。<br><br>- Material You テーマ<br>- ファーストクラスな .lrc 形式の歌詞をサポート<br>- アルバムカバーに基づいた再生中のカラーテーマ機能 (任意)<br>- SD カードに対応<br>- MediaStore を使用した音楽データベースの高速なアクセス<br>- 自然な並べ替えと様々な並べ替えオプション機能<br>- より高品質なアルバムカバーを実現する「拡張アルバムカバー」機能 (任意)<br>- 音楽のリスト内からフォルダを除外する機能をサポート<br>- 読み取り専用のプレイリストをサポート<br>- システムまたはサードパーティーのイコライザーアプリをサポート<br>- リストとグリッドビューの表示をサポート<br>- シャッフル機能は完全にランダムですべての曲を再生<br>- フォルダとファイルシステムの参照をサポート<br>- 検索に対応<br><br>何か問題がありましたら、お気軽に GitHub で問題を報告してください: <a href="https://github.com/AkaneTan/Gramophone/issues">https://github.com/AkaneTan/Gramophone/issues</a><br><br>セキュリティ関連または機密事項に関するコメントはメールからご連絡ください: nift4dev [@] gmail [.] com<br><br>Gramophone はオープンソースであり、GPL v3.0 以降のライセンスに基づいてリリースされています。すべての貢献者と翻訳者に感謝します! <a href="https://github.com/AkaneTan/Gramophone">https://github.com/AkaneTan/Gramophone</a><br><br>Gramophone の翻訳にご協力ください: <a href="https://hosted.weblate.org/engage/gramophone/">https://hosted.weblate.org/engage/gramophone/</a><br><br>注意: Gramophone には「拡張されたアルバムカバー」機能をサポートしています。この機能には「写真と動画」の権限が必要です。この機能は完全に任意であり、デフォルトでは無効化されています。データは外部にアップロードは一切行いません。画像はアルバムアートを表示するために権限が使用されます。<br><br>注意: すべてのファイル形式がすべてのデバイスで動作するわけではありません。詳細については <a href="https://developer.android.com/media/media3/exoplayer/supported-formats#progressive-container">https://developer.android.com/media/media3/exoplayer/supported-formats#progressive-container</a> および <a href="https://developer.android.com/media/platform/supported-formats#audio-formats">https://developer.android.com/media/platform/supported-formats#audio-formats</a> を参照してください。<br><br>注意: Gramophone は完全にオフラインで動作します。音楽のダウンロードやストリーミングはサポートしていません。デバイス上の音楽ファイルの再生のみに対応しています。コピー保護をバイパスすることもできません。<br><br>注意: プライバシーポリシーは <a href="https://akanefoundation.github.io/org/privacy.html">https://akanefoundation.github.io/org/privacy.html</a> からご覧いただけます。</p>
